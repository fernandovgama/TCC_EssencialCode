Oferecemos embalagens sustentáveis biodegradáveis e compostáveis. Todos os produtos seguem normas ambientais.

3. Seus Direitos e Deveres

Você tem direito a:

  • Solicitar informações e orçamentos
  • Receber atendimento de qualidade

Você deve:

Política de Privacidade

1. Quais dados coletamos?

  • Dados pessoais: Nome, e-mail, telefone, CPF/CNPJ
  • Dados de contato: Endereço completo
  • Dados de navegação: Como você usa nosso site

2. Por que coletamos?

  • Para atender suas solicitações
  • Para enviar informações relevantes
  • Para melhorar nosso site
  • Para cumprir a lei

3. Com quem compartilhamos?

Apenas com:

  • Empresas de entrega (para enviar seu pedido)
  • Autoridades (quando exigido por lei)

Não vendemos seus dados!

4. Seus direitos (LGPD)

Você pode:

Ver seus dados
Corrigir dados
Excluir dados
Cancelar consentimento

Para exercer seus direitos: privacidade@ecobytes.com

5. Segurança

Protegemos seus dados com:

  • Criptografia (conexão segura)
  • Acesso controlado
  • Backups regulares
